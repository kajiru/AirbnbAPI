AirBnB API

Unofficial pages documenting AirBnB's API

Welcome to the AirBnB API Documentation page.

This page documents the API endpoints in order to access user-logged-in and public AirBnB data. All APIs are HTTP-based and use JSON data. For questions and concerns regarding this documentation, please email nrahnemoon at gmail dot com.

Listing Search

HTTP Method: GET

Endpoint: https://api.airbnb.com/v2/search_results

Complete sample request: https://api.airbnb.com/v2/search_results?client_id=3092nxybyb0otqw18e8nh5nty&locale=en-US¤cy=USD&_format=for_search_results&_limit=10&_offset=0&guests=1&ib=false&ib_add_photo_flow=true&location=Los%20Angeles%2C%20CA&min_bathrooms=0&min_bedrooms=0&min_beds=0&min_num_pic_urls=10&mobile_session_id=&sort=1&suppress_facets=true&user_lat=37.34722222222222&user_lng=-122.04833333333333

Basic sample request: https://api.airbnb.com/v2/search_results?client_id=3092nxybyb0otqw18e8nh5nty

Default Required Parameters:

Key Value Description
client_id 3092nxybyb0otqw18e8nh5nty API Key

Required Parameters:

Other than the default required parameter above (the API key), all other parameters are optional.

Optional Parameters:

Key Sample Value Description
locale en-US Desired lagnuage
currency USD Desired currency
_format for_search_results Not sure what this is for.
_limit 10 Number of results to show.
_limit 10 Number of listings to show at a time.
_offset 0 Number of listings to offset in search.
guests 1 Number of guests.
ib false Not sure.
ib_add_photo_flow true Not sure.
location Los%20Angeles%2C%20CA Search by location name -- if unsure of lat/lng, etc.

min_bedrooms 0 Minimum number of bedrooms.
min_beds 0 Minimum number of beds.
min_num_pic_urls 10 Minimum number of pictures.

user_lng -122.42833333333333 Longitude search coordinate.

View Listing Info

HTTP Method: GET

Endpoint: https://api.airbnb.com/v2/listings/(listing ID here)

Complete sample request: https://api.airbnb.com/v2/listings/5116458?client_id=3092nxybyb0otqw18e8nh5nty&locale=en-US¤cy=USD&_format=v1_legacy_for_p3&_source=mobile_p3&number_of_guests=1

Basic sample request: https://api.airbnb.com/v2/listings/5116458?client_id=3092nxybyb0otqw18e8nh5nty&_format=v1_legacy_for_p3

Default Required Parameters:

Key Value Description
client_id 3092nxybyb0otqw18e8nh5nty API Key
_format v1_legacy_for_p3 API result format (just put this -- it won't work without it)

Required Parameters:

Other than the default required parameter above (the API key and the format), all other parameters are optional.

Optional Parameters:

Key Sample Value Description
locale en-US Desired lagnuage
_source mobile_p3 Not sure. I'm guessing this means the request is coming from an Android mobile phone.
number_of_guests 1 Determines listing availability dates based on the # of guests.

View User Info

HTTP Method: GET

Endpoint: https://api.airbnb.com/v2/users/(user ID here)

Complete sample request: https://api.airbnb.com/v2/users/2917444?client_id=3092nxybyb0otqw18e8nh5nty&_format=v1_legacy_show

Basic sample request: https://api.airbnb.com/v2/users/2917444?client_id=3092nxybyb0otqw18e8nh5nty&locale=en-US¤cy=USD&_format=v1_legacy_show

Default Required Parameters:

Key Value Description
client_id 3092nxybyb0otqw18e8nh5nty API Key
_format v1_legacy_show API result format (just put this -- it'll work without it, but it won't have as much data)

Required Parameters:

Other than the default required parameter above (the API key and the format), all other parameters are optional.

Optional Parameters:

Key Sample Value Description
locale en-US Desired lagnuage
currency USD Currency for listings.
